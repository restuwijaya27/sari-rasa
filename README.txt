Aplikasi Sederhana Penyewaan Properti

Cara installasi :
1. Buat database di mysql dengan nama 'propertiku'.
2. Import file 'propertiku.sql' di dalam database tersebut.
3. Letakkan folder project di dalam folder web service htdocs (Xampp) atau www di (Laravel)
4. Akses aplikasi di browser dengan alamat 'http://localhost/propertiku
5. Coba inputkan data sewa di halaman 'Sewa Properti'
6. Untuk melihat history pemensanan lihat di halaman 'Daftar Pesanan'

Note :
Disarankan menggunakan koneksi internet ketika membuka aplikasi, 
dikarenakan library CSS Bootstrap menggunakan versi online cdn bukan offline file.
